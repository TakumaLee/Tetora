# Tetora Roadmap

This is a high-level overview of planned features. Priorities may shift based on community feedback.

## v1.1 — Quality of Life

- [ ] Homebrew tap (`brew install TakumaLee/tap/tetora`)
- [ ] Docker image (`docker run -v ~/.tetora:/data tetora`)
- [ ] Improved onboarding wizard with web UI
- [ ] Shell completion improvements (context-aware)
- [ ] Config validation and migration tooling

## v1.2 — Platform Expansion

- [ ] Windows Service management
- [ ] Linux .rpm package
- [ ] macOS code signing and notarization
- [ ] ARM32 Linux support (Raspberry Pi)

## v1.3 — Collaboration

- [ ] Multi-user support with access control
- [ ] Shared knowledge bases
- [ ] Team dashboards
- [ ] Audit logging

## v1.4 — Advanced Agents

- [ ] Agent-to-agent delegation
- [ ] Long-running workflow orchestration
- [ ] Custom tool development SDK
- [ ] Plugin marketplace

## Future Ideas

- Mobile companion app (iOS/Android)
- Voice interaction mode
- Local model support (Ollama, llama.cpp)
- End-to-end encryption for conversations
